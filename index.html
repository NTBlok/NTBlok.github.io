Educational Web App


An educational web app to teach anatomy of human skeleton based on a native iPad app. The web app is built on the django framework with the javascript rendering engine, pixi.js. Spritesheets containing image frames of individual bones are used with json files describing the image data to render image textures as sprites. For each sprite, polygon collision areas are defined to detect touch/click on visible bone regions.



Predictive Modeling


Predictive modeling by a gradient boosted classifier (GBC), a powerful classification ensemble is used for a multi-classification problem to predict a class of reasons for denial of healthcare claims. Python scikit-learn (sklearn) library is used in an ipython notebook. Label encoding and feature scaling are performed with sklearn's Preprocessing package. Data is split into 3 sets including 60% training, 20% testing, and 20% cross-validation. Data splitting is stratified to manage an unbalanced denial reason code outcome class using the StratifiedShuffleSplit method of sklearn's Cross-Validation package. Model evaluation is conducted by constructing a confusion matrix with the final trained model and the cross-validation set using sklearn's Metrics package. The Seaborn library is used for visualization. A classifiaction report is generated to detail measures of model performance including precision, recall, f1-score, and support.



Tools Development


Use of an iPython notebook for preliminary development for a tool to output an array of coordinates marking a bounding polygon to use for a hit or touch region. The tool's argument will be an image with a transparent channel. The tool's methods will be implemented in a spritesheet generator and the output bounding polygon will be part of its resulting json data.
