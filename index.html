My Work

AI Financial Advisor

A browser extension and backend API that uses LangChain and OpenAI to provide personalized financial advice.

This project features a Chrome extension that analyzes financial data and provides personalized advice through a secure backend API. The system uses LangChain to process natural language queries and OpenAI's models to generate context-aware financial recommendations. Built with React for the frontend and FastAPI for the backend, it includes user authentication, data encryption, and real-time market data integration.

Technologies: Python, FastAPI, React, Scikit-learn, LangChain, OpenAI
