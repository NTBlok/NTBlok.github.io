AI Financial Assistant

A browser-native AI agent that observes financial dashboards, provides compliant investment advice, and executes approved actions with an audit trail.

The AI Financial Assistant is a prototype for a browser-native agent designed to assist with decisions and automate investment actions. The extension is built with TypeScript WebExtension (Manifest V3) and operates directly within the browser to observe live web-based financial dashboards. It extracts context from the DOM and sends structured snapshots to a Python FastAPI backend server for AI processing. The backend runs a local LLM using Ollama for development and privacy, which is designed to be swappable for OpenAI API or other hosted LLMs. Using the LLM's responses, the backend advises financial actions for the user such as trade recommendations. The backend logs AI reasoning and user decisions that can be stored in a database such as SQLite or PostgreSQL for later reviewing. The backend uses Playwright automation to execute user approved financial actions within the target web interface. A React dashboard displays financial data and audit logs. The dashboard supports internal data from the AI agent and can be adapted to external financial open APIs.

Technologies: TypeScript, WebExtension, Python, FastAPI, React, LangChain, Ollama/OpenAI, Playwright, SQLite/PostgreSQL

Financial Anomaly Detection System

A full stack application that detects and explains unusual patterns in stock or cryptocurrency markets using statistical modeling, machine learning and LLM-powered optimized analytics.

The Financial Anomaly Detection System is a prototype for a full-stack platform for real-time identification and interpretation of irregular patterns in financial data. The Python FastAPI backend integrates open-source financial APIs to stream and analyze live market trends for stocks or cryptocurrencies. The backend processes time-series data through a semi-supervised anomaly detection model based on an Isolation Forest model and feature-engineering using Python Scikit-learn. The React frontend is a dashboard to visualize market trends, anomaly points, and model explanations. Users can explore time-based anomalies interactively, optimize analytics with LLM assistance, and review historical data stored in the backend SQLite database. To optimize analytics and insights, a local LLM (Mistral via Ollama) interprets the machine learning model's results in natural language. The system is containerized with Docker and includes testing with Pytest and Jest.

Technologies: Python, FastAPI, React, Typescript, Scikit-learn, Ollama, Docker, Pytest, Jest

Hardware Subscription Platform

A full stack web platform for managing hardware subscriptions, monitoring device health, and predicting maintenance needs through data-driven analytics.

The Hardware Subscription Platform is a full stack system that streamlines the management of hardware device subscriptions with advanced monitoring and predictive maintenance capabilities. The frontend interface is built with Next.js and TypeScript, and the backend is powerd by Python Flask with RESTful APIs. The platform supports real-time telemetry data from connected devices storing performance and device health data in a PostgreSQL or SQLite database. The collected data is analyzed by the backend using machine learning models trained on historical hardware metrics to predict potential device failures. The interactive web dashboard provides a visual overview of performance trends and anomaly failure patterns with maintenance insights. Users can explore and manage hardware assets with AI assited recommendations and automated alerts. The platform is integrated with authentication and authorization permission role-based rules for administrators, technicians, and subscribers. The platform is containerized using Docker to support scalability.

Technologies: Python, Flask, Scikit-learn, Next, Typescript, SQLite/PostgreSQL, Docker, LLM
